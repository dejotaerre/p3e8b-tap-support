Fuse for Windows 1.6.0
======================

This is a binary distribution of Fuse (the Free Unix Spectrum Emulator) for
Windows.  For an introduction to Fuse, see README.txt.  For more detailed
documentation, read fuse.html.

Requirements
------------

Windows 2000/XP/Vista/7/8/10.

Libraries
---------

These binaries were built with:

    audiofile 0.3.6-git-b62c902
    bzip2 1.0.6
    libgcrypt 1.8.2
    libgpg-error 1.28
    libpng 1.6.34
    libspectrum 1.5.0
    libstdc++6 10.2.0
    libxml2 2.9.8
    winpthreads 8.0.0
    zlib 1.2.11
